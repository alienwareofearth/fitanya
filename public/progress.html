How to set up iOS Shortcut (one-time, 3 minutes)
1
Open Shortcuts app on your iPhone → tap + to create new shortcut → name it "Fitanya Sync"
2
Tap Add Action → search "Health" → choose Find Health Samples (or "Get Quantity")
Set: Type = Steps · Start = Start of Day · End = Now · Aggregate = Sum → rename result variable to Steps
3
Add another Health action for Active Energy Burned (calories) → variable Calories
Aggregate = Sum, same time range
4
Add another Health action for Walking + Running Distance → variable Distance
5
Add another Health action for Exercise Minutes → variable ActiveMin
6
Add Get Contents of URL action:
• Method: POST
• URL: paste your Sync URL from above
• Headers: Content-Type → application/json
• Body type: JSON
• Add keys: steps → Steps variable · calories → Calories · distance_km → Distance · active_minutes → ActiveMin · date → Current Date (formatted as yyyy-MM-dd)
7
Automate it: Go to Automation tab → New Automation → Time of Day → set to 10:00 PM daily → add "Run Shortcut" → choose "Fitanya Sync"
Turn off "Ask Before Running" so it runs silently in the background
✅ Once set up, your steps and activity will sync automatically every night and appear here on your progress page.
